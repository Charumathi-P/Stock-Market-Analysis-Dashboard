# 📈 Stock Market Analysis Dashboard
A responsive Stock Market Analysis Dashboard built using HTML, CSS, JavaScript, and Chart.js. This application allows users to search for stocks, view real-time prices, analyze historical trends, and maintain a personal watchlist.

## 🚀 Features
- 🔍 Search stocks by ticker symbol
- 📊 Interactive stock price charts
- 💰 Real-time stock prices
- 📈 Daily change and percentage gain/loss
- 🏢 Company information
- ⭐ Add/Remove stocks from Watchlist
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 💾 Local Storage support

---

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Alpha Vantage API

---

## ⚙️ Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/Stock-Market-Dashboard.git
```
2. Open the project folder.
3. Get a free API key from Alpha Vantage.
4. Replace the API key in `api.js`
```javascript
const API_KEY = "YOUR_API_KEY";
```
5. Open `index.html` in your browser.
